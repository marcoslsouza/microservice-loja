
########################## Exemplo basico de microservico ######################################

Microservico loja;

Microservico fornecedor;

Para configurar as bases de dados. Criar um repositorio no GITHUB "microservice-repo" com as pastas dos servicos "fornecedor" e "loja", com os arquivos de configuracao "fornecedor.yml" e "loja.yml";

Arquivo export Postman "Microservicos.postman_collection.json";

Banco de dados utilizado Postgresql;

Arquivo "src/main/resources/logback.xml" para configurar os logs no Papertrail com Spring Sleuth 
(para identificar os logs de uma determinada requisicao);
