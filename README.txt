
########################## Exemplo basico de microservico ######################################

Microservico loja;

Microservico fornecedor;

Microservico transportador;

Para configurar as bases de dados. Criar um repositorio no GITHUB "microservice-repo" com as pastas dos servicos "fornecedor", "loja" e "transportador", com os arquivos de configuracao "fornecedor.yml", "loja.yml" e "transportador.yml";

Arquivo export Postman "Microservicos.postman_collection.json";

Banco de dados utilizado Postgresql;
